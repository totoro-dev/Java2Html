


申明方法：m：表示方法名；P：表示参数类型；p：表示参数名
	1. 声明方法必须指定访问限制：private，protected，public；
	2. 第一行总是以' { '符号结尾，方法逻辑代码另起新行
	3. 无参方法：(注意空格）方法名和左括号之间不留空格，其它最多一个空格
		3.1  m(){  或  m() {
		3.2  m( ){  或  m( ) {
	4. 有参方法：在无参方法的规范上，添加多个参数的规范（注意空格）
		4.1  m(P p){  或 m(P p) {
		4.2  m(P p,P p){  或  m(P p, P p){  ......
		4.2  m(P p,P p){  或  m(P p, P p){  ......
